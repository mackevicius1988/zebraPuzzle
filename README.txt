Application is written in Java 1.7, so you need Java Runtime Enviroment to run this app, it can be downloaded from 
http://www.oracle.com/technetwork/java/javase/downloads/jre7-downloads-1880261.html
To run the application launch run.cmd command file

The solution is provided in the form of a Maven (http://maven.apache.org/download.cgi) project. It can
be built by invoking the run.cmd command file.

The input of the program is in input.csv, to change the input just modify the input file and re-run the application.
